# Modulora

**Discover your next great component.**

Modulora is a creator-first component registry for developers and coding agents. Browse open and commercial components from trusted creators, install with one command, or let your coding agent handle it.

## Repositories

- [`modulora`](https://github.com/Modulora/modulora) — self-hostable public registry core
- [`spec`](https://github.com/Modulora/spec) — open manifests, provenance, evidence, and API contracts
- [`cli`](https://github.com/Modulora/cli) — secure component discovery and installation

The project is currently in its planning and foundation stage.
